## 📚 设计决策档案

> 已完成的决策，仅保留结论。详细过程见 Git 历史。

### 真相源架构 ✅
- **决策**：append-only event log 是真相源；`story_state.json` 是可重建投影
- **时间**：2026-06-09
- **影响**：崩溃恢复、版本分叉、状态重放均基于 event log

### 最小数据模型 ✅
- **决策**：V1 只做 5 个核心对象（Character / PlotEvent / Fact / ContinuityEvent / project_preferences）
- **时间**：2026-06-15
- **文档**：`openspec/changes/define-minimal-story-state-and-event-log/`

### Agent 架构收敛 ✅
- **决策**：核心 4（对话/提取/记忆/矛盾监控）+ 按需能力 + 策略器/门禁
- **时间**：2026-06-06
- **文档**：`AGENTS.md`「架构收敛」段

### 知识图谱定位 ✅
- **决策**：LightRAG 用于灵感检索/低频报告，不进实时矛盾判断主链路
- **原因**：对内向型叙事（心理描写、情感驱动）效果有限
- **时间**：2026-06-07

### 首发人群 ✅
- **决策**：单人故事创作者/编剧，需要连续性提醒
- **放弃**：团队协作、多人实时编辑、生产流程管理
- **时间**：2026-06-17
